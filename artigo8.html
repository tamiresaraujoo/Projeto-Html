    • Tik Tok, a nova mídia alternativa para jovens para compartilhamento de talentos online: um estudo analítico

    Resumo : Tik Tok, a nova onda da mídia social que introduziu uma nova subcultura para a geração atual e trouxe uma revolução nas mídias sociais para os jovens. Tik Tok tornou-se uma nova plataforma para retratar talentos para jovens, adicionando um novo significado às plataformas de mídia social já existentes. É a mídia social de crescimento mais rápido que fascinou e envolveu tantos jovens em todo o mundo e é uma das que mais cresce na Índia. Tik Tok é basicamente a nova versão alternativa de compartilhamento online, que permite a seus usuários sua interface para preparar vídeos curtos e peças musicais. Tik Tok é a nova obsessão entre os jovens que está dando a eles uma plataforma para expressá-los de forma inovadora e engenhosa. O presente trabalho é baseado no estudo de caso de indivíduos que usaram o Tik Tok para destacar seu talento. Como citar este artigo: Patel K, Binjola H. Tik Tok, a nova mídia alternativa para jovens para compartilhamento de talentos online: um estudo analítico.
