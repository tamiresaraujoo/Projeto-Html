    • Assistente virtual para gerar respostas personalizadas dentro de uma sessão de comunicação

    Resumo : : Agentes inteligentes (IA) para geração automática de respostas ao conteúdo dentro de uma sessão de comunicação (CS) são divulgados. Um IA é treinado para direcionar as respostas a um usuário e ao contexto do usuário dentro do CS. Um IA recebe conteúdo CS que inclui expressões de linguagem natural que codificam as conversas dos usuários e determina recursos de conteúdo com base em modelos de linguagem natural. Os recursos de conteúdo indicam a semântica pretendida das expressões. O IA identifica o conteúdo provavelmente relevante para o usuário-alvo, para gerar uma resposta. Identificar tal conteúdo inclui determinar uma relevância do conteúdo com base em recursos de conteúdo, um contexto do CS, um modelo de interesse do usuário e um modelo de relevância de conteúdo. A identificação do conteúdo provavelmente relevante para responder é baseada na relevância determinada do conteúdo e nos limites de relevância.
